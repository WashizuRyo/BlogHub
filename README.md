# The-View
  GitHubで認証でき、ブログを書いたり、閲覧できたりするアプリケーションです。<br>
  レスポンシブ対応のためスマートフォンからもご確認いただけます。<br>

# 使用技術
  - HTML
    - pug
  - CSS
    - Bootstrap
  - JavaScript
    - Node.js
    - Express
  - Postgresql
  - Docker

# 機能一覧
  - GitHubでのログイン機能(login.js)
  - ブログ作成,編集,削除,機能(blogs.js)
  - コメント機能(comments.js)
    - コメント編集(fetch)
